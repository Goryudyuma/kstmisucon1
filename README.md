# API Server

Simple Rest API using gin(framework) & gorm(orm)

## Endpoint list

### Comments Resource

```
GET    /comments
GET    /comments/:id
POST   /comments
PUT    /comments/:id
DELETE /comments/:id
```

### Users Resource

```
GET    /users
GET    /users/:id
POST   /users
PUT    /users/:id
DELETE /users/:id
```

server runs at http://localhost:8080
